Main code is stored in unification.cpp, it's a C++ code.

To compile and run the code, you can use the following commands in your terminal:
    g++ unification.cpp -o unify
    ./unify

You can input 2 terms, once you are done typing 1st term then press enter and like wise after 2nd term. Spacing will be handled by the code, you can chose to enter space if you would like after comma.
